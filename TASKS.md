# 反重力飞行对战游戏任务 To-Do

## 1. 使用方式

- `Status` 取值：`TODO` / `DOING` / `BLOCKED` / `DONE`
- `Owner` 先留空，开发时按人分配
- `Depends On` 用于标记前置任务
- 每完成一个任务，应补充链接到代码、录屏或测试记录

## 5. 阻塞项清单

以下问题一旦发生，优先处理：

- [ ] 固定步长更新后手感与高帧率表现不一致
- [ ] 四元数姿态更新出现抖动或翻转
- [ ] 漂移模型导致无法瞄准
- [ ] 速度特效明显影响命中判断
- [ ] 粒子数量稍增就出现明显掉帧
